    CSS Positioning

    There are four methods for positioning elements

    1. block: everything gets its own line, takes up full width of page
    2. inline-block: the above, except elements can sit together
    3. inline:makes the element sit on the same line as another element, but without formatting it like a block. It only takes up as much width as it needs (not the whole line).
    4. none: makes content disappear
    1. The margin is the space around the element. A sample value is auto, which centers the element, but there are also margins for top, bottom, left and right. They can be set in the order top, right, bottom, left, i.e 50px 20px 30px 40px
    2. The border is what the edge of the element looks like. It can be input in the order: size display-type hex-color, i.e. 40px dashed #FF0000
    3. The padding is the space in between the content and the border, which can be made smaller or bigger. Again, values are input in top,right,bottom,left order, or if all four sides are the same, you can declare one value. Can have negative values to move in the opposite direction
    4. Content is the element itself

    Floats position where elements go on the page. Simply: float:right

    clear: pushes other elements out of the way. Left clears everything to the left, right etc, both will clear everything on the left and right

    Absolute, relative and fixed positioning

    Elements default to static positioning if they are not styled. Absolute elements are positioned in reference to the first object that is not static.
